*便利な機能
C = ctrlキー / M = Altキー

C + b : 後方に1文字分移動
C + f : 前方に1文字分移動
C + a : 行頭に移動
C + e : 行末に移動

M + b : 後方に単語1つ分移動
M + f : 前方に単語1つ分移動

C + w : 後方にスペース区切りで1単語分を削除

C + k : カーソル位置から行末までを削除
C + u : カーソル位置から行頭までを削除
C + y : 最後に削除した内容を挿入

C + s : 画面表示ロック
C + q : 画面表示ロック解除

C + l : 画面を消去

C + r : 履歴をインクリメンタル検索
C + g : 検索結果を破棄


*コマンド
  pwd : カレントディレクトリを表示
  cd : カレントディレクトリを変更
  ls : ディレクトリ内のファイルを表示

  lsコマンド
    * : 任意の文字列 ex)ls ba*  //  ls *.html
    ? : 任意の1文字 ex)ls ba??

      オプション
      -l : ファイルの詳細情報表示
      -a : すべてのファイルを表示
      -F : ファイルの種類を表す季語を追加して表示
      -w : 指定した数値の横幅で表示
      --quote-name : ファイル名をダブルクォーテーションで囲んで表示
      --quote

  mkdirコマンド
    ディレクトリを作成

      オプション
      -p : 深いディレクトリを一気に作成

  touchコマンド
    ファイルを作成

    rm / rmdirコマンド
    ファイル/ディレクトリを削除

    rm ファイル名
    rm ファイル名1 ファイル名2 ...

      オプション
      -r : ディレクトリツリー削除 ex) rm -r ディレクトリ名
      -i : 削除確認

    rmdir ディレクトリ名（ディレクトリの中身が空である条件）

  catコマンド
    ファイルを表示
    cat パス
    cat のみの場合入力 → C + dで戻る


      オプション
      -n : 行番号を表示
  
  lessコマンド
    スクロール表示
      スクロール表示時のコマンド
        スペースキー, f, C + v : 1画面下にスクロール
        b, M + v : 1画面上にスクロール
        j, C + n, enter : 1行下にスクロール
        k, C + p : 1行上にスクロール
        q : lessコマンドを終了
        /文字列 : 下方向に向かって文字を検索
        ?文字列 : 上方向に向かって文字を検索
        n : 次の検索結果に移動
        N : 前の検索結果の移動
        
  lnコマンド
    リンクを張る
      ex) ln オプション リンク先ファイル名 リンク名
      
      オプション
        -s : シンボリックリンクを作成
