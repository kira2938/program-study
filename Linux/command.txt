*便利な機能
C = ctrlキー / M = Altキー

C + b : 後方に1文字分移動
C + f : 前方に1文字分移動
C + a : 行頭に移動
C + e : 行末に移動

M + b : 後方に単語1つ分移動
M + f : 前方に単語1つ分移動

C + w : 後方にスペース区切りで1単語分を削除

C + k : カーソル位置から行末までを削除
C + u : カーソル位置から行頭までを削除
C + y : 最後に削除した内容を挿入

C + s : 画面表示ロック
C + q : 画面表示ロック解除

C + l : 画面を消去

C + r : 履歴をインクリメンタル検索
C + g : 検索結果を破棄


*コマンド
  pwd : カレントディレクトリを表示
  cd : カレントディレクトリを変更
  ls : ディレクトリ内のファイルを表示

  /===== lsコマンド
    * : 任意の文字列 ex)ls ba*  //  ls *.html
    ? : 任意の1文字 ex)ls ba??

      オプション
      -l : ファイルの詳細情報表示
      -a : すべてのファイルを表示
      -F : ファイルの種類を表す季語を追加して表示
      -w : 指定した数値の横幅で表示
      --quote-name : ファイル名をダブルクォーテーションで囲んで表示
      --quote

  /===== mkdirコマンド
    ディレクトリを作成

      オプション
      -p : 深いディレクトリを一気に作成

  /===== touchコマンド
    ファイルを作成

    rm / rmdirコマンド
    ファイル/ディレクトリを削除

    rm ファイル名
    rm ファイル名1 ファイル名2 ...

      オプション
      -r : ディレクトリツリー削除 ex) rm -r ディレクトリ名
      -i : 削除確認

    rmdir ディレクトリ名（ディレクトリの中身が空である条件）

  /===== catコマンド
    ファイルを表示
    cat パス
    cat のみの場合入力 → C + dで戻る


      オプション
      -n : 行番号を表示
  
  /===== lessコマンド
    スクロール表示
      スクロール表示時のコマンド
        スペースキー, f, C + v : 1画面下にスクロール
        b, M + v : 1画面上にスクロール
        j, C + n, enter : 1行下にスクロール
        k, C + p : 1行上にスクロール
        q : lessコマンドを終了
        /文字列 : 下方向に向かって文字を検索
        ?文字列 : 上方向に向かって文字を検索
        n : 次の検索結果に移動
        N : 前の検索結果の移動
        
  /===== lnコマンド
    リンクを張る
      ex) ln オプション リンク先ファイル名 リンク名
      
      オプション
        -s : シンボリックリンクを作成

  /===== findコマンド
    ディレクトリツリーからファイルを探す ex)find 検索開始ディレクトリ 検索条件 アクション $find . -name file-1.txt -print
      ファイルの種別を-typeで指定
        -type f : ファイル
        -type d : ディレクトリ
        -type l : シンボリックリンク
        ex) $ find . -type d -print        
        
      オプション
        -a : 複数の検索条件の指定
        ex) $ find . -type f -a -name '*.txt' -print
        
  /===== locateコマンド
    ファイル名データベースからファイルを探す
    ex) $ locate オプション 検索パターン
    
      bashという文字列を含むパス名を検索
        $ locate bash
        
      拡張子.sedのファイルを検索
        $ locate '*.sed'
        
      大文字小文字を区別しない
        $ locate -i notes
        
      ファイル名だけを対象に検索
        $ locate -b python
        
      複数の検索パターンを指定するとOR検索になる
        $ locate docs document
        
      複数の検索パターンすべてを含むようAND検索する
        $ locate -A bash doc
        
  /===== manコマンド
    マニュアルを表示  ex) $ man 調べたいコマンド名
    
    オプション
      -k : キーワードで調べる
      
      コマンド(N) Nは1-9でこの番号はセッションというもの
        1 : コマンド
        2 : システムコール
        3 : ライブラリ関数
        4 : デバイスファイル
        5 : ファイル書式
        6 : ゲーム
        7 : その他いろいろなもの
        8 : システム管理コマンド
        9 : カーネルルーチン
        
    /===== whichコマンド
      コマンドフルパスを表示
      
      オプション
        -a : すべてのサーチパスから検索
        
    /===== 日本語ドキュメントと英語ドキュメント
      日本語に指定してヘルプを表示
        $ LANG=ja_JP.UTF-8 cat --help
        
    /===== Vim    $ vim ファイル名
      デフォルトのエディタ
      
      vimのコメント
        :q  vimを終了
        :w  ファイルを上書き保存
        :q! ファイルを保存せずに終了
        x   カーソル位置の文字を削除
        i   カーソル位置の左に文字を追加
        a   カーソル位置の右に文字を追加
        w   前方に単語1つ分移動
        b   後方に単語1つ分移動
        W   スペース区切りで前方に単語1つ分移動
        B   スペース区切りで後方に端豪1つ分移動
        O   行頭に移動
        $   行末に移動
        gg  1行目に移動
        G   最後の行に移動
        数字G N行目に移動
        d$  行末まで削除
        d0  行頭まで削除
        dw  単語1つを削除
        dgg 最初の行まで削除
        dG 最後の行まで削除
        yy  現在カーソルのある行をヤンク
        dd  現在カーソルのある行を削除
        j   下の行を連絡
        u   元に戻す
        C + r 取り消す
        /文字列  下方に文字列を検索
        ?文字列  上方に文字列を検索
        n   次の検索結果に移動
        N   前の検索結果に移動
        %s/置換元文字列/置換後文字列/g  文字列の置換
